Map Pathfinding Visualizer

This app was developed to visualize how pathfinding algorithms search through a real-life map.

Currently only supports A* search, but I'll be adding more pathfinding algorithms in the future

Created using WebGL

Github

Instructions:

  • Click and hold to pan
  • Scroll to zoom in/out
  • Click to set starting point
  • Click again to set destination point
  • Once animation is completed, clicking will reset the map and set the starting point
